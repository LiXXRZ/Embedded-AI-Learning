# 连续LTI系统：零输入响应 ($y_{zi}(t)$) 复习笔记

## 1. 核心概念
*   **定义**：当输入信号 $x(t)=0$ 时，仅由系统的**初始状态**（如电容电压初始值、电感电流初始值）所产生的响应。
*   **物理意义**：系统内部储能释放的过程（例如RC电路的电容放电）。
*   **数学本质**：求解齐次微分方程的通解。

## 2. 数学模型与求解步骤
系统的微分方程一般形式为：
$$y^{(n)}(t) + a_{n-1}y^{(n-1)}(t) + \dots + a_0 y(t) = b_m x^{(m)}(t) + \dots$$

**求解零输入响应的三个步骤：**
1.  **列特征方程**：令微分方程右边为0，将 $y^{(n)}(t)$ 替换为 $s^n$，得到特征方程 $s^n + a_{n-1}s^{n-1} + \dots + a_0 = 0$。
2.  **定通解形式**：求出特征根 $s_i$，根据根的类型（单实根、重根、复根）确定 $y_{zi}(t)$ 的函数形式。
3.  **求待定系数**：代入初始条件（$y(0^-), y'(0^-)\dots$），解方程组求出系数 $K_i$。

---

## 3. 三种特征根情况详解与例题

### 情况一：特征根为不等的实根 ($s_1 \neq s_2$)
**通解形式**：
$$y_{zi}(t) = K_1 e^{s_1 t} + K_2 e^{s_2 t}$$

> **【例题 1】**
> **方程**：$y''(t) + 6y'(t) + 8y(t) = x(t), \quad t>0$
> **初始状态**：$y(0^-)=1, \quad y'(0^-)=2$
>
> **求解过程**：
> 1.  **特征方程**：$s^2 + 6s + 8 = 0 \Rightarrow (s+2)(s+4) = 0$
> 2.  **特征根**：$s_1 = -2, \quad s_2 = -4$
> 3.  **通解形式**：$y_{zi}(t) = K_1 e^{-2t} + K_2 e^{-4t}$
> 4.  **代入初值**：
>     *   $y(0) = K_1 + K_2 = 1$
>     *   $y'(t) = -2K_1 e^{-2t} - 4K_2 e^{-4t} \Rightarrow y'(0) = -2K_1 - 4K_2 = 2$
> 5.  **解方程组**：联立上述两式，解得 $K_1 = 3, \quad K_2 = -2$
>
> **结果**：$y_{zi}(t) = 3e^{-2t} - 2e^{-4t}, \quad t \ge 0$

---

### 情况二：特征根为重实根 ($s_1 = s_2 = s$)
**通解形式**：
$$y_{zi}(t) = (K_1 + K_2 t)e^{st} = K_1 e^{st} + K_2 t e^{st}$$

> **【例题 2】**
> **方程**：$y''(t) + 4y'(t) + 4y(t) = 3x(t), \quad t>0$
> **初始状态**：$y(0^-)=2, \quad y'(0^-)=-1$
>
> **求解过程**：
> 1.  **特征方程**：$s^2 + 4s + 4 = 0 \Rightarrow (s+2)^2 = 0$
> 2.  **特征根**：$s_1 = s_2 = -2$ (二重根)
> 3.  **通解形式**：$y_{zi}(t) = K_1 e^{-2t} + K_2 t e^{-2t}$
> 4.  **代入初值**：
>     *   $y(0) = K_1 = 2$
>     *   求导（注意乘积法则）：$y'(t) = -2K_1 e^{-2t} + K_2(1 \cdot e^{-2t} + t \cdot (-2)e^{-2t})$
>     *   $y'(0) = -2K_1 + K_2 = -1$
> 5.  **解方程组**：将 $K_1=2$ 代入，得 $-4 + K_2 = -1 \Rightarrow K_2 = 3$
>
> **结果**：$y_{zi}(t) = 2e^{-2t} + 3te^{-2t}, \quad t \ge 0$

---

### 情况三：特征根为共轭复根 ($s_{1,2} = \sigma \pm j\omega$)
**通解形式**：
$$y_{zi}(t) = e^{\sigma t}(K_1 \cos\omega t + K_2 \sin\omega t)$$

> **【例题 3】**
> **方程**：$y''(t) + 2y'(t) + 5y(t) = 4x(t), \quad t>0$
> **初始状态**：$y(0^-)=1, \quad y'(0^-)=3$
>
> **求解过程**：
> 1.  **特征方程**：$s^2 + 2s + 5 = 0$
> 2.  **特征根**：$s = \frac{-2 \pm \sqrt{4-20}}{2} = -1 \pm j2$
>     *   实部 $\sigma = -1$，虚部 $\omega = 2$
> 3.  **通解形式**：$y_{zi}(t) = e^{-t}(K_1 \cos 2t + K_2 \sin 2t)$
> 4.  **代入初值**：
>     *   $y(0) = e^0(K_1 \cdot 1 + 0) \Rightarrow K_1 = 1$
>     *   求导：$y'(t) = -e^{-t}(K_1 \cos 2t + K_2 \sin 2t) + e^{-t}(-2K_1 \sin 2t + 2K_2 \cos 2t)$
>     *   $y'(0) = -(K_1) + 2K_2 = 3$
> 5.  **解方程组**：将 $K_1=1$ 代入，得 $-1 + 2K_2 = 3 \Rightarrow 2K_2 = 4 \Rightarrow K_2 = 2$
>
> **结果**：$y_{zi}(t) = e^{-t}(\cos 2t + 2\sin 2t), \quad t \ge 0$

---

## 4. 易错点总结
1.  **微分方程右边设为0**：求解 $y_{zi}(t)$ 时，不管原方程右边的输入 $x(t)$ 是什么，一律视为 0。
2.  **求导运算**：在代入 $y'(0)$ 时，尤其是遇到**重根**（$te^{st}$）和**复根**（衰减震荡）的情况，求导需要使用乘积法则，非常容易算错系数。
    *   *复根求导技巧*：对于 $y(t) = e^{\sigma t}(K_1 \cos \omega t + K_2 \sin \omega t)$，有 $y'(0) = \sigma K_1 + \omega K_2$。记忆这个结论可以加速计算。
3.  **适用范围**：此方法仅适用于 $t \ge 0$ 的部分（或者说 $t \ge 0^-$）。
