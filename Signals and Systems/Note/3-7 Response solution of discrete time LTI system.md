# 学习笔记：离散LTI系统的响应与卷积计算

## 一、 零输入响应 (Zero Input Response, $y_{zi}[k]$)
**定义**：输入信号为零，完全由系统的**初始状态**单独作用产生的响应。
**数学模型**：齐次差分方程 $\sum_{i=0}^n a_i y_{zi}[k-i] = 0$

### 求解步骤与特征根对应的解形式：
**Step 1**：写出系统的特征方程，求出**特征根 (Feature roots)** $r$。
**Step 2**：根据特征根的类型，设出零输入响应的形式（重点背诵）：
1.  **不相等的实根 (Unequal solid roots)** $r_1, r_2, \dots, r_n$：
    $$y_{zi}[k] = C_1 r_1^k + C_2 r_2^k + \dots + C_n r_n^k$$
2.  **相等的实重根 (Equal solid roots)** 比如 $r_1 = r_2 = r$：
    $$y_{zi}[k] = C_1 r^k + C_2 k r^k$$
    *(注：与连续系统重根乘 $t$ 类似，离散系统重根乘 $k$)*
3.  **共轭复根 (Conjugate compound roots)** $r_{1,2} = a \pm jb = \rho e^{\pm j\Omega_0}$：
    $$y_{zi}[k] = C_1 \rho^k \cos(\Omega_0 k) + C_2 \rho^k \sin(\Omega_0 k)$$

**Step 3**：代入系统给定的初始状态（如 $y[-1], y[-2]$ 等），解出待定系数 $C_1, C_2$。

#### 📝 例题 1：特征根为“不相等的实根” (Two unequal solid roots)
**【题目】** 已知系统差分方程 $y[k] + 3y[k-1] + 2y[k-2] = x[k], (k \ge 0)$，初始状态 $y[-1]=0, y[-2]=1/2$。求零输入响应 $y_{zi}[k]$。
**【详细解答】**
1.  **写出特征方程**：令 $y[k] \to r^2, y[k-1] \to r, y[k-2] \to 1$
    $$r^2 + 3r + 2 = 0$$
2.  **求特征根**：$(r+1)(r+2) = 0 \implies r_1 = -1, r_2 = -2$
3.  **设齐次解形式**：
    $$y_{zi}[k] = C_1(-1)^k + C_2(-2)^k$$
4.  **代入初始状态求系数**（⚠️ PPT跳过的计算细节）：
    *   代入 $k = -1$：$C_1(-1)^{-1} + C_2(-2)^{-1} = 0 \implies -C_1 - \frac{1}{2}C_2 = 0 \implies C_1 = -\frac{1}{2}C_2$
    *   代入 $k = -2$：$C_1(-1)^{-2} + C_2(-2)^{-2} = \frac{1}{2} \implies C_1 + \frac{1}{4}C_2 = \frac{1}{2}$
    *   联立解方程：将 $C_1$ 代入第二个式子，$-\frac{1}{2}C_2 + \frac{1}{4}C_2 = \frac{1}{2} \implies -\frac{1}{4}C_2 = \frac{1}{2} \implies \mathbf{C_2 = -2}$。从而 $\mathbf{C_1 = 1}$。
5.  **最终结果**：
    $$y_{zi}[k] = (-1)^k - 2(-2)^k, \quad k \ge 0$$

#### 📝 例题 2：特征根为“相等的实重根” (Two equal solid roots)
**【题目】** 已知 $y[k] + 4y[k-1] + 4y[k-2] = x[k], (k \ge 0)$，初始状态 $y[-1]=0, y[-2]=1/2$。求 $y_{zi}[k]$。
**【详细解答】**
1.  **特征方程**：$r^2 + 4r + 4 = 0 \implies (r+2)^2 = 0 \implies r_1 = r_2 = -2$
2.  **设齐次解形式**（⚠️ 注意重根要乘 $k$）：
    $$y_{zi}[k] = C_1 k(-2)^k + C_2(-2)^k$$
3.  **代入初始状态求系数**：
    *   代入 $k = -1$：$C_1(-1)(-2)^{-1} + C_2(-2)^{-1} = 0 \implies \frac{C_1}{2} - \frac{C_2}{2} = 0 \implies \mathbf{C_1 = C_2}$
    *   代入 $k = -2$：$C_1(-2)(-2)^{-2} + C_2(-2)^{-2} = \frac{1}{2} \implies -\frac{2C_1}{4} + \frac{C_2}{4} = \frac{1}{2} \implies -\frac{C_1}{2} + \frac{C_2}{4} = \frac{1}{2}$
    *   联立解方程：因为 $C_1=C_2$，代入得 $-\frac{C_1}{2} + \frac{C_1}{4} = \frac{1}{2} \implies -\frac{C_1}{4} = \frac{1}{2} \implies \mathbf{C_1 = -2, C_2 = -2}$。
4.  **最终结果**：
    $$y_{zi}[k] = -2k(-2)^k - 2(-2)^k, \quad k \ge 0$$

#### 📝 例题 3：特征根为“共轭复根” (Conjugate compound roots)
**【题目】** 已知 $y[k] + 2y[k-1] + 2y[k-2] = x[k], (k \ge 0)$，初始状态 $y[-1]=0, y[-2]=1/2$。求 $y_{zi}[k]$。
**【详细解答】**
1.  **特征方程**：$r^2 + 2r + 2 = 0$
2.  **求特征根（化为极坐标）**：利用求根公式 $r = \frac{-2 \pm \sqrt{4 - 8}}{2} = -1 \pm j$。
    转为极坐标：模长 $\rho = \sqrt{(-1)^2 + 1^2} = \sqrt{2}$，幅角 $\Omega_0 = \frac{3\pi}{4}$。即 $r_{1,2} = \sqrt{2}e^{\pm j \frac{3\pi}{4}}$。
3.  **设齐次解形式**：
    $$y_{zi}[k] = (\sqrt{2})^k \left( C_1 \cos\frac{3\pi}{4}k + C_2 \sin\frac{3\pi}{4}k \right)$$
4.  **代入初始状态求系数**：
    *   代入 $k = -1$（利用 $\cos(-\frac{3\pi}{4}) = -\frac{\sqrt{2}}{2}, \sin(-\frac{3\pi}{4}) = -\frac{\sqrt{2}}{2}$）：
        $y[-1] = \frac{1}{\sqrt{2}}[C_1(-\frac{\sqrt{2}}{2}) + C_2(-\frac{\sqrt{2}}{2})] = -\frac{1}{2}C_1 - \frac{1}{2}C_2 = 0 \implies C_1 = -C_2$
    *   代入 $k = -2$（利用 $\cos(-\frac{3\pi}{2}) = 0, \sin(-\frac{3\pi}{2}) = 1$）：
        $y[-2] = \frac{1}{2}[C_1(0) + C_2(1)] = \frac{1}{2}C_2 = \frac{1}{2} \implies \mathbf{C_2 = 1}$，从而 $\mathbf{C_1 = -1}$。
5.  **最终结果**：
    $$y_{zi}[k] = (\sqrt{2})^k \left[ -\cos\frac{3\pi}{4}k + \sin\frac{3\pi}{4}k \right], \quad k \ge 0$$

## 二、 单位冲激响应 (Unit Impulse Response, $h[k]$)
**定义**：在**系统初始状态为零**的条件下，输入信号为单位脉冲序列 $\delta[k]$ 时，系统产生的输出响应。

### 求解方法：等效初始条件法 (Equivalent Initial Condition Method)
课件中给出了求 $h[k]$ 的一套标准流程：
1.  **方程转换**：将原差分方程右侧换成 $\delta[k]$。
    例：$h[k] + 3h[k-1] + 2h[k-2] = \delta[k]$
2.  **求等效初始条件**：
    由于因果系统在 $k < 0$ 时 $h[k]=0$（即 $h[-1]=h[-2]=0$），代入原方程递推求出 $k=0$ 和 $k=1$ 时的值：
    *   $k=0$: $h[0] + 3(0) + 2(0) = \delta[0] = 1 \implies \mathbf{h[0] = 1}$
    *   $k=1$: $h[1] + 3(1) + 2(0) = \delta[1] = 0 \implies \mathbf{h[1] = -3}$
3.  **转为求齐次解**：
    当 $k > 0$ 时，$\delta[k] = 0$。此时等价于求满足初始条件 $h[0]=1, h[1]=-3$ 的齐次差分方程的解。
    设 $h[k] = C_1(-1)^k + C_2(-2)^k$，代入 $h[0], h[1]$ 解得 $C_1=-1, C_2=2$。
4.  **得出结果**（别忘了乘上 $u[k]$ 保证因果性）：
    $$h[k] = [-(-1)^k + 2(-2)^k]u[k]$$

#### 📝 例题 4：等效初始条件法
**【题目】** 因果离散系统的差分方程为 $y[k] + 3y[k-1] + 2y[k-2] = x[k]$。求冲激响应 $h[k]$。
**【详细解答】**
1.  **确定 $h[k]$ 的齐次解形式**：
    特征方程 $r^2+3r+2=0 \implies r_1=-1, r_2=-2$。
    设 $h[k] = C_1(-1)^k + C_2(-2)^k \quad (k \ge 0)$。
2.  **寻找等效初始条件 $h[0], h[1]$**：
    将原方程写为 $h[k] + 3h[k-1] + 2h[k-2] = \delta[k]$。由于是因果系统，当 $k<0$ 时 $h[k]=0$（即 $h[-1]=h[-2]=0$）。
    *   令 $k=0$：$h[0] + 3(0) + 2(0) = \delta[0] = 1 \implies \mathbf{h[0] = 1}$
    *   令 $k=1$：$h[1] + 3h[0] + 2(0) = \delta[1] = 0 \implies h[1] + 3(1) = 0 \implies \mathbf{h[1] = -3}$
3.  **代入求解系数**：
    *   $h[0] = C_1 + C_2 = 1$
    *   $h[1] = -C_1 - 2C_2 = -3$
    两式相加：$-C_2 = -2 \implies \mathbf{C_2 = 2}$，从而 $\mathbf{C_1 = -1}$。
4.  **最终结果**（乘以 $u[k]$ 保证因果性）：
    $$h[k] = [-(-1)^k + 2(-2)^k]u[k]$$

## 三、 零状态响应与离散卷积和 (Zero-state Response & Convolution Sum)
**核心公式**：零状态响应 $y_{zs}[k]$ 等于输入信号 $x[k]$ 与冲激响应 $h[k]$ 的离散卷积和：
$$y_{zs}[k] = x[k] * h[k] = \sum_{n=-\infty}^{\infty} x[n]h[k-n]$$

### 卷积的三种计算方法：
#### 1. 解析法 (Parsing method)
直接代入求和公式，利用等比数列求和等数学技巧进行计算。
*   **经典公式背诵**：若 $x[k] = \alpha^k u[k]$， $h[k] = \beta^k u[k]$，则：
    *   当 $\alpha \ne \beta$ 时： $x[k] * h[k] = \frac{\beta^{k+1} - \alpha^{k+1}}{\beta - \alpha} u[k]$
    *   当 $\alpha = \beta$ 时： $x[k] * h[k] = (k+1)\alpha^k u[k]$

#### 📝 例题 5：解析法直接求和 (Parsing Method)
**【题目】** 计算卷积 $x[k] * h[k]$，其中 $x[k] = \alpha^k u[k]$， $h[k] = \beta^k u[k]$。
**【详细解答】**
1.  **列出卷积公式**：$y[k] = \sum_{n=-\infty}^{+\infty} \alpha^n u[n] \cdot \beta^{k-n} u[k-n]$
2.  **确定求和上下限**：
    $u[n]$ 限制了 $n \ge 0$；$u[k-n]$ 限制了 $k-n \ge 0 \implies n \le k$。
    所以求和区间变为 $n \in [0, k]$（前提是 $k \ge 0$，所以最终结果要乘 $u[k]$）。
3.  **进行等比数列求和**：
    $$y[k] = \sum_{n=0}^{k} \alpha^n \beta^{k-n} = \beta^k \sum_{n=0}^{k} \left(\frac{\alpha}{\beta}\right)^n$$
4.  **分类讨论结果**：
    *   当 $\alpha \ne \beta$ 时（利用等比求和公式 $S_n = \frac{1-q^{n+1}}{1-q}$）：
        $$y[k] = \beta^k \frac{1 - (\alpha/\beta)^{k+1}}{1 - \alpha/\beta} u[k] = \frac{\beta^{k+1} - \alpha^{k+1}}{\beta - \alpha} u[k]$$
    *   当 $\alpha = \beta$ 时（公比为1，直接相加 $k+1$ 次）：
        $$y[k] = (k+1)\alpha^k u[k]$$

#### 2. 图解法 (Graphical method)
常用于只有图形没有公式的有限长序列。
**操作五步曲**：换元 (变量变为 $n$) $\to$ 翻转 ($h[n]$ 变 $h[-n]$) $\to$ 平移 (移动 $k$ 步) $\to$ 对应点相乘 $\to$ 所有乘积相加。
> 💡 **关键结论：序列长度与起点**
> 卷积结果的**起点** = 原两序列起点之和。
> 卷积结果的**终点** = 原两序列终点之和。
> 卷积结果的**长度** = $L_1 + L_2 - 1$。

#### 📝 例题 6：图解法计算卷积 (Graphical Method)
**【题目】** 已知 $x[k]$ 是一个幅度为1，长度为4的方波（$k=0,1,2,3$）；$h[k]$ 是幅度为1，长度为3的方波（$k=0,1,2$）。求 $y[k] = x[k]*h[k]$。
**【详细解答】** 
将 $h[n]$ 翻转得到 $h[-n]$，然后向右平移 $k$ 个单位，观察与 $x[n]$ 的重叠点数（因为幅值都是1，重叠了几个点，乘积之和就是几）：
*   **(a) $k < 0$**：两波形无交集 $\implies y[k] = 0$
*   **(b) $0 \le k < 2$**：$h$ 正在进入 $x$。重叠点数为 $k+1$ $\implies y[k] = k+1$ （即 $y[0]=1, y[1]=2$）
*   **(c) $2 \le k \le 3$**：$h$ 完全包含在 $x$ 中。始终有3个点重叠 $\implies y[k] = 3$ （即 $y[2]=3, y[3]=3$）
*   **(d) $3 < k \le 5$**：$h$ 正在离开 $x$。重叠点数为 $6-k$ $\implies y[k] = 6-k$ （即 $y[4]=2, y[5]=1$）
*   **(e) $k > 5$**：两波形彻底分离 $\implies y[k] = 0$
**综合结果**：$y[k] = \{1, 2, 3, 3, 2, 1\}$ （序列起点为 $k=0$）。

#### 3. 列表法 / 对角线相加法 (List method)
**极其适合考试中的手算！**
将 $h[k]$ 的值排成一行，将 $x[k]$ 的值排成一列。两两相乘填满表格。**沿着左下到右上的对角线把数值加起来**，就是卷积结果。
*   **课件例题**：$x[k]=\{1, 2, 0, 3, 2\}$ （下划线在第1个元素，即起点为 $k=-2$）；$h[k]=\{1, 4, 2, 3\}$（起点为 $k=0$）。
*   列表计算出的新序列为 $\{1, 6, 10, 10, 20, 14, 13, 6\}$。
*   **新起点定位**：$k = (-2) + 0 = -2$。所以结果从 $k=-2$ 的位置开始排。

#### 📝 例题 7：列表法计算卷积 (List Method)
**【题目】** 计算 $x[k] = \{1, 2, \underline{0}, 3, 2\}$ 与 $h[k] = \{\underline{1}, 4, 2, 3\}$ 的卷积（下划线代表 $k=0$ 的位置）。
**【详细解答】**
1.  **定位新起点**：$x[k]$ 起点是 -2，$h[k]$ 起点是 0。卷积结果的起点是 $-2 + 0 = -2$。
2.  **画表乘法**（横排 $x$，竖排 $h$）：

| $h \backslash x$ | 1 ($x_{-2}$) | 2 ($x_{-1}$) | 0 ($x_{0}$) | 3 ($x_{1}$) | 2 ($x_{2}$) |
| :---- | :-- | :-- | :-- | :-- | :-- |
| **1 ($h_0$)**   | 1   | 2   | 0   | 3   | 2   |
| **4 ($h_1$)**   | 4   | 8   | 0   | 12  | 8   |
| **2 ($h_2$)**   | 2   | 4   | 0   | 6   | 4   |
| **3 ($h_3$)**   | 3   | 6   | 0   | 9   | 6   |

1.  **对角线相加**（从左上到右下，代表相同的 $k$ 值）：
    *   $k=-2$: **1**
    *   $k=-1$: 4 + 2 = **6**
    *   $k=0$: 2 + 8 + 0 = **10** （这就是结果的 $k=0$ 位置）
    *   $k=1$: 3 + 4 + 0 + 3 = **10**
    *   $k=2$: 6 + 0 + 12 + 2 = **20**
    *   $k=3$: 0 + 6 + 8 = **14**
    *   $k=4$: 9 + 4 = **13**
    *   $k=5$: **6**
2.  **最终结果**：$y[k] = \{1, 6, \underline{10}, 10, 20, 14, 13, 6\}$。

## 四、 离散卷积的性质 (Properties of Convolution)
类似于乘法，卷积满足多种代数规律，可以极大简化计算：

1.  **交换律 (Commutative law)**: $x[k] * h[k] = h[k] * x[k]$
2.  **结合律 (Binding law)**: $(x * h_1) * h_2 = x * (h_1 * h_2)$
3.  **分配律 (Distributive law)**: $x * (h_1 + h_2) = x * h_1 + x * h_2$
4.  **移位特性 (Displacement)**: 
    *   如果 $x[k]*h[k] = y[k]$，那么 $x[k-n] * h[k-l] = y[k-(n+l)]$。
    *   **最重要性质**：与 $\delta[k-n]$ 卷积等于**移位** $\implies x[k] * \delta[k-n] = x[k-n]$。
5.  **差分与求和特性 (Difference & Summation)**:
    *   $\nabla x[k] * h[k] = x[k] * \nabla h[k] = \nabla y[k]$
    *   $(\sum x[n]) * h[k] = \sum y[n]$

### 📝 例题 8：利用卷积性质巧算 (Properties of Convolution)
**【题目】** 已知 $h[k] = \{\underline{1}, 4, 5, 3\}$， $x[k] = \{1, 0, \underline{2}, 4\}$（下划线为 $k=0$）。求 $y[k] = x[k]*h[k]$。
**【详细解答】**
1.  **分解 $x[k]$ 为冲激信号**：
    $x[k]$ 在 $k=-2$ 处为1，在 $k=0$ 处为2，在 $k=1$ 处为4。
    即：$x[k] = \delta[k+2] + 2\delta[k] + 4\delta[k-1]$
2.  **利用分配律和移位性质**：
    由于 $\delta[k-n] * h[k] = h[k-n]$，我们将卷积展开：
    $y[k] = h[k+2] + 2h[k] + 4h[k-1]$
3.  **执行序列移位与相加**（严格对齐 $k$ 坐标进行列竖式计算）：
    *   **$h[k+2]$**（左移2位，起点变为 $k=-2$）： `{1, 4, 5, 3, 0, 0, 0}`
    *   **$2h[k]$**（原位乘2，起点 $k=0$）：`{0, 0, 2, 8, 10, 6, 0}`
    *   **$4h[k-1]$**（右移1位并乘4，起点 $k=1$）：`{0, 0, 0, 4, 16, 20, 12}`
4.  **按列求和**：
    * $k=-2$: 1
    * $k=-1$: 4
    * $k=0$: 5 + 2 + 0 = $\mathbf{7}$ (此为新的 $k=0$ 点)
    * $k=1$: 3 + 8 + 4 = 15
    * $k=2$: 0 + 10 + 16 = 26
    * $k=3$: 0 + 6 + 20 = 26
    * $k=4$: 0 + 0 + 12 = 12
5.  **最终结果**：$y[k] = \{1, 4, \underline{7}, 15, 26, 26, 12\}$。
