# 课程笔记：连续时间信号的基本运算

## 一、 时域变换 (Time Domain Transformations)
这是信号分析的基础，主要涉及自变量 $t$ 的改变。

### 1. 核心变换类型
*   **反转 (Reversal)**: $x(t) \to x(-t)$
    *   以纵轴 ($t=0$) 为中心，将波形左右翻转 $180^\circ$。
*   **尺度变换 (Compression/Extension)**: $x(t) \to x(at), a>0$
    *   $a > 1$: **压缩** (Compression)。波形变窄，播放速度变快。
    *   $0 < a < 1$: **扩展** (Extension)。波形变宽，播放速度变慢。
*   **时移 (Shifting)**: $x(t) \to x(t \pm t_0), t_0>0$
    *   $x(t - t_0)$: **右移** (延迟, Delay)。
    *   $x(t + t_0)$: **左移** (超前, Advance)。
    *   *口诀：左加右减*。

### 2. 组合运算 (难点与技巧)
当一个信号同时包含缩放、反转和平移时（例如从 $x(t)$ 变换为 $x(6-2t)$），有两种常用方法：

#### 方法一：逐步变换法 (Step-by-step)
*   **顺序很重要**。通常建议先进行压缩/反转，最后进行平移。
*   **示例** : 画出 $x(6-2t)$
    1.  **压缩**: $t \to 2t$, 得到 $x(2t)$。
    2.  **反转**: $t \to -t$, 得到 $x(-2t)$。
    3.  **平移**: 注意此时 $t$ 的系数不为1。要凑出 $6-2t = -2(t-3)$。
        *   因此是将 $t$ 替换为 $t-3$。
        *   即 **右移 3个单位**。

#### 方法二：端点函数值不变法 (Endpoint Function Value Invariance Method)
*   **核心思想**: 信号的变换只是自变量的线性映射，波形上的关键点（起点、终点、折点）对应的函数值不变。
*   **公式**:
    设已知信号自变量为 $t_{old}$，目标信号自变量为 $t_{new}$。
    令 **旧变量整体 = 新变量整体**，求解 $t_{new}$。
    $$t_{old} = a \cdot t_{new} + b$$
*   **高阶例题** :
    *   **已知**: $x(3t+2)$ 的波形（起点 $t_{old}=-2$，终点 $t_{old}=3$）。
    *   **求画**: $x(6-2t)$ 的波形。
    *   **映射计算**:
        令 $3t_{old} + 2 = 6 - 2t_{new}$ (这里逻辑需修正，应为**已知函数的自变量表达式 = 目标函数的自变量表达式**？不，应是**波形上的特征点对应**)。
        
        **更正的理解（逻辑）**:
        我们要找 $x(6-2t)$ 的关键点。
        已知波形是 $f(t) = x(3t+2)$。
        *   原图中 $t=-2$ 处对应的是 $x(3\times(-2)+2) = x(-4)$ 的值。
        *   原图中 $t=3$ 处对应的是 $x(3\times3+2) = x(11)$ 的值。
        这个思路比较绕。
        
        **推荐解法 (最稳健)**:
        利用中间变量 $u$ 代表 $x(\cdot)$ 括号里的内容。
        1.  原图是 $x(3t+2)$。
            *   起点 $t=-2 \implies u = 3(-2)+2 = -4$。即 $x(u)$ 从 $u=-4$ 开始。
            *   终点 $t=3 \implies u = 3(3)+2 = 11$。即 $x(u)$ 到 $u=11$ 结束。
        2.  目标是 $x(6-2t)$。
            *   令 $6-2t = -4$ (起点) $\implies 2t=10 \implies t=5$。
            *   令 $6-2t = 11$ (终点) $\implies 2t=-5 \implies t=-2.5$。
        3.  结论：目标波形分布在 $[-2.5, 5]$ 之间。

## 二、 代数运算

*   **相加 (Addition)**: $x_1(t) + x_2(t)$，同一时刻幅值相加。
*   **相乘 (Multiplication)**: $x_1(t) \cdot x_2(t)$。
    *   **物理意义**: 调制 (Modulation)、加窗 (Windowing)。
    *   **应用**: 调幅广播 (AM)，即 $y(t) = x(t)\cos(\omega_c t)$。

## 三、 微积分运算 (重点：涉及冲激信号)

### 1. 微分 (Differentiation)
*   **规则**:
    *   连续部分：直接求导。
    *   **跳变点 (间断点)**：导数变为 **冲激信号 (Impulse, $\delta(t)$)**。
    *   冲激强度 (面积) = 跳变高度 ($Right - Left$)。

### 2. 积分 (Integration)
*   $y(t) = \int_{-\infty}^{t} x(\tau) d\tau$
*   脉冲的积分是斜坡；冲激的积分是阶跃。

### 3. 重要推导例题
**题目**: 已知 $x(t) = \sin(\omega_0 t)[u(t) - u(t-T)]$，其中 $\omega_0 = 2\pi/T$ (即 $T$ 为一个完整周期)。求 $x'(t)$ 和 $x''(t)$。

#### 第一步：求一阶导 $x'(t)$
利用乘积求导法则 $(uv)' = u'v + uv'$。
$$x'(t) = [\omega_0 \cos \omega_0 t] \cdot [u(t) - u(t-T)] + [\sin \omega_0 t] \cdot [\delta(t) - \delta(t-T)]$$

*   **化简技巧 (利用冲激信号的筛选性)**:
    *   $f(t)\delta(t) = f(0)\delta(t)$
    *   $\sin(\omega_0 t)\delta(t) = \sin(0)\delta(t) = 0$
    *   $\sin(\omega_0 t)\delta(t-T) = \sin(\omega_0 T)\delta(t-T) = \sin(2\pi)\delta(t-T) = 0$
*   **结果**:
    $$x'(t) = \omega_0 \cos \omega_0 t [u(t) - u(t-T)]$$
    *(物理意义：正弦波完整周期切片，首尾值为0，无跳变，故导数无冲激)*

#### 第二步：求二阶导 $x''(t)$
对 $x'(t)$ 再次求导：
$$x''(t) = \frac{d}{dt} \{ \omega_0 \cos \omega_0 t [u(t) - u(t-T)] \}$$
$$= [-\omega_0^2 \sin \omega_0 t] \cdot [u(t) - u(t-T)] + [\omega_0 \cos \omega_0 t] \cdot [\delta(t) - \delta(t-T)]$$

*   **化简**:
    *   $\omega_0 \cos(\omega_0 t)\delta(t) = \omega_0 \cos(0)\delta(t) = \omega_0 \delta(t)$
    *   $\omega_0 \cos(\omega_0 t)\delta(t-T) = \omega_0 \cos(2\pi)\delta(t-T) = \omega_0 \delta(t-T)$
*   **结果**:
    $$x''(t) = \omega_0 \delta(t) - \omega_0 \delta(t-T) - \omega_0^2 \sin \omega_0 t [u(t) - u(t-T)]$$
    *(物理意义：余弦波在 $0$ 和 $T$ 处有跳变，因此二阶导出现了成对的冲激信号)*

## 四、 信号的分解与表示 (Signal Representation)

### 1. 用基本信号表示复杂波形
利用阶跃信号 $u(t)$ 和斜变信号 $r(t)$ 可以合成各种分段线性波形。

*   **阶跃信号 $u(t)$**: 用于表示“开关”或矩形脉冲。
    *   例如：在 $t=0$ 开始， $t=1$ 结束的矩形 = $u(t) - u(t-1)$。
*   **斜变信号 $r(t)$**: 用于表示斜率的变化。
    *   $r(t) = t \cdot u(t)$。
*   **合成技巧 (Slide 23-24)**:
    1.  从左向右观察波形。
    2.  **出现拐点**：说明斜率发生了变化。
    3.  **斜率变化量 $\Delta k$**: 加一个 $ \Delta k \cdot r(t - t_0)$。
    4.  **出现跳变**: 加一个 $Height \cdot u(t - t_0)$。

### 2. 例题分析
波形描述：
*   $t < -1$: 值为 0 (斜率 0)
*   $t = -1$: 开始上升，斜率为 1 $\rightarrow$ 加 $r(t+1)$
*   $t = 0$: 变平，斜率从 1 变为 0 (变化量 -1) $\rightarrow$ 减 $r(t)$
*   $t = 1$: 向下跳变，幅度为 1 $\rightarrow$ 减 $u(t-1)$
*   **综合表达式**:
    $$x(t) = r(t+1) - r(t) - u(t-1)$$
