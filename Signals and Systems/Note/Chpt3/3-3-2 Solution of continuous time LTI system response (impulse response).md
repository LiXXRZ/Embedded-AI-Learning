# 连续 LTI 系统冲激响应 $h(t)$ 复习笔记

## 一、 冲激响应的定义 (Definition)

### 1. 物理定义
*   **条件**：系统初始状态为 **零** (Zero State)。
*   **激励**：输入信号为单位冲激信号 $\delta(t)$。
*   **响应**：此时系统的零状态响应即为冲激响应，记为 $h(t)$。
    $$ \delta(t) \longrightarrow \boxed{\text{连续LTI系统}} \longrightarrow h(t) $$

### 2. 特性
*   $h(t)$ 反映了连续 LTI 系统的时域特性。
*   它仅取决于系统本身的结构和参数（特征根），与输入信号无关。

### 3. 数学描述
若系统由线性常系数微分方程描述：
$$ y^{(n)}(t) + \dots + a_0 y(t) = b_m x^{(m)}(t) + \dots + b_0 x(t) $$
则 $h(t)$ 满足：
$$ h^{(n)}(t) + \dots + a_0 h(t) = b_m \delta^{(m)}(t) + \dots + b_0 \delta(t) $$
*   **注意**：当 $t > 0$ 时，$\delta(t)$ 及其导数均为 0，方程变为齐次方程。因此，$t > 0$ 时 $h(t)$ 的形式由齐次解（特征根）决定。

---

## 二、 冲激响应的求解：冲激平衡法 (Impulse Balancing Method)

这是求解 $h(t)$ 的核心方法，通过平衡方程两边的奇异函数项（$\delta(t)$ 及其导数）来确定系数。

### 1. 判别阶数
设微分方程左端（输出 $y$）最高阶数为 $n$，右端（输入 $x$）最高阶数为 $m$。

### 情况 A：$n > m$ (真分式系统)
*   **特点**：$h(t)$ 中不包含 $\delta(t)$ 及其导数项，仅由特征根决定的模态项组成。但其**前 $(n-m-2)$ 阶导数在 $t=0$ 处是连续的（为0）**，而第 $(n-m-1)$ 阶及以上的导数在 $t=0$ 处会发生跳变。

*   **待定解形式**：
    $$h(t) = \left( \sum_{i=1}^{n} C_i e^{s_i t} \right) u(t)$$
    *(注：若有重根，形式需按齐次解规则调整，如 $(C_1 + C_2 t)e^{st} u(t)$)*
*   **解题步骤**：
    1.  求特征根 $s_i$。
    2.  设 $h(t) = A e^{s t} u(t)$。
    3.  代入微分方程，利用 $\delta(t)$ 平衡系数。

    > **【例题 1】** $y'(t) + 3y(t) = 2x(t)$
    > 1.  **特征根**：$s + 3 = 0 \Rightarrow s = -3$。
    > 2.  **设解**：$h(t) = A e^{-3t} u(t)$。
    > 3.  **求导**：$h'(t) = -3A e^{-3t} u(t) + A e^{-3t} \delta(t) = -3A e^{-3t} u(t) + A \delta(t)$。
    > 4.  **代入方程**：
    >     $[ -3A e^{-3t} u(t) + A \delta(t) ] + 3 [ A e^{-3t} u(t) ] = 2 \delta(t)$
    >     化简得：$A \delta(t) = 2 \delta(t) \Rightarrow A = 2$。
    > 5.  **结果**：$h(t) = 2 e^{-3t} u(t)$。

    > **【例题 2】 $n=2, m=0$ 的情况**
    > 已知系统的微分方程为：$y''(t) + 3y'(t) + 2y(t) = x(t)$，求其冲激响应 $h(t)$。
    >
    > **1. 特征根**：$s^2 + 3s + 2 = 0 \Rightarrow (s+1)(s+2) = 0 \Rightarrow s_1 = -1, s_2 = -2$。 
    > **2. 设解**：$h(t) = (A e^{-t} + B e^{-2t}) u(t)$。
    > **3. 逐阶求导（关键步）**：
    > *   $h'(t) = (-A e^{-t} - 2B e^{-2t}) u(t) + \underbrace{(A + B) e^0 \delta(t)}_{\text{此处设为 } 0}$
    >     *注：为了使 $h''(t)$ 中最高阶冲激项与右边 $\delta(t)$ 平衡，低阶导数处的 $\delta$ 项系数必须能支撑到最高阶。*
    >     根据冲激平衡原则，若 $n=2, m=0$，则 $h(0_+) = 0$，即 **$A+B = 0$**。
    >     此时：$h'(t) = (-A e^{-t} - 2B e^{-2t}) u(t)$
    > *   $h''(t) = (A e^{-t} + 4B e^{-2t}) u(t) + (-A - 2B) \delta(t)$
    >
    > **4. 代入方程平衡系数**：
    > 将 $h''(t), h'(t), h(t)$ 代入 $h'' + 3h' + 2h = \delta(t)$：
    > $[ (A e^{-t} + 4B e^{-2t}) u(t) + (-A - 2B) \delta(t) ] + 3 [ (-A e^{-t} - 2B e^{-2t}) u(t) ] + 2 [ (A e^{-t} + B e^{-2t}) u(t) ] = \delta(t)$
    >
    > *   **指数项（齐次项）**：自动抵消（因为 $s$ 是特征根）。
    > *   **$\delta(t)$ 项系数**：$-A - 2B = 1$
    >
    > **5. 解方程组**：
    > $\begin{cases} A + B = 0 \\ -A - 2B = 1 \end{cases} \Rightarrow \begin{cases} A = 1 \\ B = -1 \end{cases}$
    >
    > **6. 结果**：$h(t) = (e^{-t} - e^{-2t}) u(t)$。

    >**【例题 3】 $n=2, m=1$ 的情况**
    >已知系统的微分方程为：$y''(t) + 5y'(t) + 6y(t) = x'(t) + 4x(t)$，求其冲激响应 $h(t)$。
    >
    >**1. 特征根**
    >特征方程：$s^2 + 5s + 6 = 0 \Rightarrow (s+2)(s+3) = 0$
    >特征根：$s_1 = -2, s_2 = -3$。
    >
    >**2. 设待定解**
    >由于 $n=2, m=1$，设 $h(t) = (A e^{-2t} + B e^{-3t}) u(t)$。
    >
    >**3. 逐阶求导（利用冲激平衡法）**
    >*   **一阶导数**：
    $$h'(t) = (-2A e^{-2t} - 3B e^{-3t}) u(t) + \underbrace{(A + B)}_{h(0_+)} \delta(t)$$
    >*   **二阶导数**：
    $$h''(t) = (4A e^{-2t} + 9B e^{-3t}) u(t) + (-2A - 3B) \delta(t) + (A + B) \delta'(t)$$
    >
    >**4. 代入方程平衡系数**
    >将 $h'', h', h$ 代入方程 $h'' + 5h' + 6h = \delta' + 4\delta$。
    >我们重点提取等号左边的奇异函数项（$\delta'$ 和 $\delta$）：
    >
    >*   **$\delta'(t)$ 项系数平衡**：
    左边只有 $h''$ 贡献了 $(A + B) \delta'$。
    $$A + B = 1 \quad \text{—— (方程①)}$$
    *(注：这对应了右边的 $x'$ 的系数 $1$)*
    >
    >*   **$\delta(t)$ 项系数平衡**：
    左边由 $h''$ 的 $(-2A - 3B)$ 和 $5h'$ 的 $5(A + B)$ 组成。
    $$(-2A - 3B) + 5(A + B) = 4 \quad \text{—— (方程②)}$$
    *(注：这对应了右边的 $4x$ 的系数 $4$)*
    >
    >**5. 解方程组**
    >由方程①得 $A + B = 1$。
    >代入方程②：
    >$(-2A - 3B) + 5(1) = 4$
    >$-2A - 3B = -1$
    >$2A + 3B = 1$
    >
    >联立：
    >$\begin{cases} A + B = 1 \\ 2A + 3B = 1 \end{cases} \Rightarrow \begin{cases} 2A + 2B = 2 \\ 2A + 3B = 1 \end{cases} \Rightarrow B = -1, A = 2$
    >
    >**6. 结果**
    >$$h(t) = (2e^{-2t} - e^{-3t}) u(t)$$

---

### 情况 B：$n \le m$ (假分式系统)
*   **特点**：方程右边出现了输入的高阶导数，导致 $h(t)$ 中不仅有指数项，还必须包含 $\delta(t)$ 及其导数项，才能平衡方程两边。
*   **通解形式**：
    $$ h(t) = \underbrace{\left( \sum_{i=1}^{n} K_i e^{s_i t} \right) u(t)}_{\text{固有分量}} + \underbrace{\sum_{j=0}^{m-n} A_j \delta^{(j)}(t)}_{\text{冲激项}} $$
*   **解题步骤**：需假设 $h(t)$ 包含 $B\delta(t)$ 等项，代入方程平衡系数。

    > **【例题 4】** $y'(t) + 3y(t) = 2x(t) + x'(t)$
    > *   这里 $n=1, m=1$，即 $n=m$，故 $h(t)$ 中包含 $\delta(t)$。
    > 1.  **特征根**：$s = -3$。
    > 2.  **设解**：$h(t) = A e^{-3t} u(t) + B \delta(t)$。
    > 3.  **求导 $h'(t)$**：
    >     利用公式 $[f(t)\delta(t)]' = f(0)\delta'(t) + \dots$，
    >     $h'(t) = (-3A e^{-3t} u(t) + A \delta(t)) + B \delta'(t)$。
    > 4.  **代入方程** $h' + 3h = 2\delta + \delta'$：
    >     $\left[ -3A e^{-3t} u(t) + A \delta(t) + B \delta'(t) \right] + 3 \left[ A e^{-3t} u(t) + B \delta(t) \right] = 2\delta(t) + \delta'(t)$
    > 5.  **合并同类项**：
    >     $\delta'(t)$ 项系数：$B = 1$
    >     $\delta(t)$ 项系数：$A + 3B = 2 \Rightarrow A + 3 = 2 \Rightarrow A = -1$
    > 6.  **结果**：$h(t) = -e^{-3t} u(t) + \delta(t)$。

    > **【例题 5】 $n=2, m=2$ 的情况**
    > 已知系统：$y''(t) + 4y'(t) + 3y(t) = 2x''(t) + x'(t) + x(t)$，求 $h(t)$。
    >
    > **1. 特征根**：$s^2 + 4s + 3 = 0 \Rightarrow (s+1)(s+3) = 0 \Rightarrow s_1 = -1, s_2 = -3$。
    > **2. 设解**：$h(t) = (A e^{-t} + B e^{-3t}) u(t) + D \delta(t)$。
    > **3. 逐阶求导**：
    > *   $h(t) = (A e^{-t} + B e^{-3t}) u(t) + D \delta(t)$
    > *   $h'(t) = (-A e^{-t} - 3B e^{-3t}) u(t) + (A+B) \delta(t) + D \delta'(t)$
    > *   $h''(t) = (A e^{-t} + 9B e^{-3t}) u(t) + (-A-3B) \delta(t) + (A+B) \delta'(t) + D \delta''(t)$
    >
    > **4. 代入方程平衡系数**：
    > 将各阶导数代入 $h'' + 4h' + 3h = 2\delta'' + \delta' + \delta$。
    >
    > 重点观察右端的奇异函数项：
    > *   **$\delta''(t)$ 项**：$D = 2$
    > *   **$\delta'(t)$ 项**：$(A + B) + 4(D) = 1 \Rightarrow A + B + 8 = 1 \Rightarrow A + B = -7$
    > *   **$\delta(t)$ 项**：$(-A - 3B) + 4(A + B) + 3(D) = 1$
    >     代入 $A+B = -7$ 和 $D=2$：
    >     $-A - 3B + 4(-7) + 3(2) = 1$
    >     $-A - 3B - 28 + 6 = 1 \Rightarrow -A - 3B = 23$
    >
    > **5. 解方程组**：
    > $\begin{cases} A + B = -7 \\ -A - 3B = 23 \end{cases} \Rightarrow -2B = 16 \Rightarrow B = -8, A = 1$
    >
    > **6. 结果**：$h(t) = (e^{-t} - 8e^{-3t}) u(t) + 2\delta(t)$。

    >**【例题 6】 $n=1, m=2$ 的情况**
    >已知系统的微分方程为：$y'(t) + 2y(t) = x''(t) + x'(t) + 3x(t)$，求 $h(t)$。
    >
    >**1. 特征根**：
    >$s + 2 = 0 \Rightarrow s = -2$。
    >
    >**2. 设待定解**：
    >因为 $m-n = 2-1 = 1$，所以 $h(t)$ 最高含有 $\delta'(t)$ 项。
    >设 $h(t) = A e^{-2t} u(t) + B \delta'(t) + C \delta(t)$。
    >
    >**3. 求导**：
    >*   $h(t) = A e^{-2t} u(t) + B \delta'(t) + C \delta(t)$
    >*   $h'(t) = -2A e^{-2t} u(t) + A \delta(t) + B \delta''(t) + C \delta'(t)$
    >
    >**4. 代入方程平衡系数**：
    >将 $h'(t)$ 和 $h(t)$ 代入 $h'(t) + 2h(t) = \delta''(t) + \delta'(t) + 3\delta(t)$：
    >$$ [ -2A e^{-2t} u(t) + A \delta(t) + B \delta''(t) + C \delta'(t) ] + 2 [ A e^{-2t} u(t) + B \delta'(t) + C \delta(t) ] = \delta''(t) + \delta'(t) + 3\delta(t) $$
    >
    >*   **$\delta''(t)$ 项系数**：$B = 1$
    >*   **$\delta'(t)$ 项系数**：$C + 2B = 1 \Rightarrow C + 2(1) = 1 \Rightarrow C = -1$
    >*   **$\delta(t)$ 项系数**：$A + 2C = 3 \Rightarrow A + 2(-1) = 3 \Rightarrow A = 5$
    >
    >**5. 结果**：
    >$$ h(t) = 5e^{-2t} u(t) + \delta'(t) - \delta(t) $$

## 补充：快速判断表

| 阶数关系 | $h(t)$ 是否含 $\delta(t)$ | $h(t)$ 的最高奇异项 | 备注 |
| :--- | :--- | :--- | :--- |
| **$n > m$** | 否 | 无 (仅含 $u(t)$) | 常见物理系统，低通特性 |
| **$n = m$** | 是 | $A \delta(t)$ | 包含比例环节 |
| **$n < m$** | 是 | $A \delta^{(m-n)}(t)$ | 包含微分环节，工程难实现 |

## 三、 总结：通用公式 (General Formulas)

连续 LTI 系统的微分方程：
$$ \sum_{k=0}^{n} a_k y^{(k)}(t) = \sum_{k=0}^{m} b_k x^{(k)}(t) $$

1.  **当 $n > m$ (且特征根为单实根)**：
    $$ h(t) = \left( \sum_{i=1}^{n} K_i e^{s_i t} \right) u(t) $$
    *   系数 $K_i$ 通过冲激平衡法求得。

2.  **当 $n \le m$**：
    $$ h(t) = \left( \sum_{i=1}^{n} K_i e^{s_i t} \right) u(t) + \sum_{j=0}^{m-n} A_j \delta^{(j)}(t) $$
    *   $u(t)$ 前的指数形式由特征方程的根决定。
    *   $\delta(t)$ 及其导数项由方程两边最高阶导数的平衡决定。

---

## 四、 核心考点提示
1.  **不要忘记 $u(t)$**：$h(t)$ 是因果信号，必须写上 $u(t)$ 或注明 $t>0$。
2.  **求导细节**：在对 $A e^{st} u(t)$ 求导时，根据乘积法则，会产生 $A e^{st} \delta(t)$，即 $A \delta(t)$。这是冲激平衡法的关键来源。
3.  **判断是否有 $\delta(t)$**：观察微分方程右边 $x(t)$ 的最高阶导数阶数是否 $\ge$ 左边 $y(t)$ 的最高阶数。如果是，解中必含 $\delta$。

---

## 五、 拔高技巧：奇异函数项的快速判断

对于 $n \ge m$ 的系统，如何快速确定 $h(t)$ 中 $\delta$ 的项？

1.  **若 $m < n$**：$h(t)$ 不含 $\delta(t)$ 项。
2.  **若 $m = n$**：$h(t)$ 含有 $D \delta(t)$ 项，且系数 $D = \frac{b_n}{a_n}$（即左右两端最高阶项系数之比）。
    *   *例如例4中，$b_2=2, a_2=1$，所以 $D=2/1=2$。*
3.  **若 $m = n+1$**：$h(t)$ 含有 $D_1 \delta'(t) + D_0 \delta(t)$。

---

## 六、 重点总结：冲激平衡法三步走 (高阶通用)

1.  **写出齐次解形式**：根据特征根列出指数项（注意 $u(t)$）。
2.  **补齐奇异项**：对比 $n$ 和 $m$，若 $n=m$ 补 $D\delta(t)$，若 $n < m$ 补更高阶导数（考试极少出现 $m > n$）。
3.  **求导并“对号入座”**：
    *   求导时务必运用 $(f(t)u(t))' = f'(t)u(t) + f(0)\delta(t)$。
    *   将 $\delta^{(k)}(t)$ 的系数在等式两边一一对应列方程。
    *   **解方程顺序**：通常从最高阶奇异项（如 $\delta^{(m-n)}$）开始解，依次向下。
