# 学习笔记：冲激响应与系统特性
**(Impulse Response Represents System Characteristics)**

## 一、 核心概念：冲激响应 $h(t)$
*   **物理意义**：对于连续时间系统，冲激响应 $h(t)$ 用于表示系统在时域（Time Domain）的特性。在系统处于零状态时，若输入信号为单位冲激函数 $\delta(t)$，此时系统的输出即为冲激响应 $h(t)$。
*   **唯一性**：不同的系统具有不同的冲激响应。只要知道了一个连续线性时不变（LTI）系统的冲激响应，就能完全描述该系统的特性。

## 二、 常见系统的冲激响应
以下是几种常见理想系统的输入与对应冲激响应：
1.  **无失真传输系统 (Distortion free transmission system)**
    *   冲激响应：$h(t) = K\delta(t - t_d)$
    *   *注：$K$ 为常数（幅值增益），$t_d$ 为输入信号通过系统后的延迟时间。*
2.  **理想积分器 (Ideal integrator)**
    *   冲激响应：$h(t) = u(t)$ 
    *   *注：$u(t)$ 为单位阶跃函数。*
3.  **理想微分器 (Ideal differentiator)**
    *   冲激响应：$h(t) = \delta'(t)$

## 三、 复合系统的冲激响应
对于由多个子系统组合而成的复杂系统，其总冲激响应可以通过子系统冲激响应的**卷积**或**相加**来求得。

### 1. 级联系统 (Cascade System)
*   **结构**：类似于集线器级联，信号依次通过系统1 $h_1(t)$ 和系统2 $h_2(t)$。
*   **过程推导**：
    *   中间信号 $z(t) = x(t) * h_1(t)$
    *   输出信号 $y(t) = z(t) * h_2(t) = x(t) *[h_1(t) * h_2(t)]$
*   **结论**：级联系统的等效冲激响应为各子系统冲激响应的**卷积**。
    $$h(t) = h_1(t) * h_2(t)$$

### 2. 并联系统 (System in Parallel)
*   **结构**：类似于计算机并联，输入信号同时进入系统1 $h_1(t)$ 和系统2 $h_2(t)$，最终输出为两支路输出之和。
*   **过程推导**：
    *   支路1输出 $y_1(t) = x(t) * h_1(t)$
    *   支路2输出 $y_2(t) = x(t) * h_2(t)$
    *   总输出 $y(t) = y_1(t) + y_2(t) = x(t) * [h_1(t) + h_2(t)]$
*   **结论**：并联系统的等效冲激响应为各子系统冲激响应的**代数和**。
    $$h(t) = h_1(t) + h_2(t)$$

### 3. 📝 【例题：求解复合系统冲激响应】
**题目**：求图中连续系统的冲激响应 $h(t)$。
已知：$h_1(t) = e^{-3t}u(t)$， $h_2(t) = \delta(t - 1)$， $h_3(t) = u(t)$。
*   **系统结构解析**：
    系统分为三条并联支路（参考课件第8页完整图）：
    1.  上方支路：$h_1(t)$ 与 $h_2(t)$ 级联，冲激响应为 $h_1(t) * h_2(t)$。
    2.  中间支路：输入直通输出，相当于经过一个 $h(t) = \delta(t)$ 的系统。
    3.  下方支路：系统 $h_3(t)$。
*   **计算过程**：
    $$h(t) = h_1(t) * h_2(t) + h_3(t) + \delta(t)$$

    $$= \delta(t - 1) * [e^{-3t}u(t)] + u(t) + \delta(t)$$

    利用冲激函数的延时抽样性质（卷积移位）：
    $$= e^{-3(t-1)}u(t-1) + u(t) + \delta(t)$$

## 四、 冲激响应与系统因果性 (Causality)
*   **因果系统定义**：系统在 $t_0$ 时刻的输出，仅取决于 $t_0$ 时刻及其之前的输入信号。换言之，**系统的输出不会超前于输入出现**。
*   **充分必要条件**：对于连续 LTI（线性时不变）系统，成为因果系统的充要条件是冲激响应在 $t<0$ 时恒为零：
    $$h(t) = 0, \quad t < 0$$

## 五、 冲激响应与系统稳定性 (Stability)
*   **稳定系统定义 (BIBO稳定)**：如果系统对于任意有界输入（Bounded Input），都会产生有界的输出（Bounded Output），则称该系统为稳定系统。
*   **充分必要条件**：对于连续 LTI 系统，成为稳定系统的充要条件是冲激响应绝对可积：
    $$\int_{-\infty}^{\infty} |h(\tau)| d\tau = S < +\infty$$

## 六、 📝 【例题：判断系统的因果性与稳定性】
**题目**：已知某连续 LTI 系统的冲激响应为 $h(t) = e^{at}u(t)$，判断该系统是否为因果系统和稳定系统。

**解答**：
1.  **判断因果性**：
    因为存在 $u(t)$（阶跃函数），当 $t < 0$ 时，$u(t) = 0$。
    因此满足 $h(t) = 0, \, t < 0$。
    **结论**：该系统是一个**因果系统**。
2.  **判断稳定性**：
    计算绝对积分：
    $$\int_{-\infty}^{\infty} |h(\tau)| d\tau = \int_{0}^{\infty} e^{a\tau} d\tau = \left. \frac{1}{a}e^{a\tau} \right|_0^{\infty}$$
    讨论参数 $a$ 的取值：
    *   **当 $a < 0$ 时**：$e^{a\infty} \to 0$，积分结果为 $-\frac{1}{a}$（有限值）。满足绝对可积条件。**系统稳定**。
    *   **当 $a \ge 0$ 时**：积分结果趋近于无穷大 $\infty$。不满足绝对可积条件。**系统不稳定**。
