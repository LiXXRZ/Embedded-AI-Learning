# 学习笔记：离散系统的单位冲激响应与特性

## 一、 核心概念
对于离散 LTI 系统，**单位冲激响应 $h[k]$**（在零状态下，输入为单位脉冲 $\delta[k]$ 时的输出）能够完全表征该系统在时域中的所有特性。只要知道了 $h[k]$，系统就“没有秘密”了。

## 二、 四种常见离散系统的 $h[k]$
我们需要像背乘法口诀一样记住这四种基本系统的冲激响应：

1.  **无失真传输系统 (Distortion-free transmission system)**
    *   物理意义：信号经过该系统后，形状不变（仅幅度缩放 $K$ 倍），且整体延迟了 $k_d$ 个采样点。
    *   冲激响应：$h[k] = K \cdot \delta[k - k_d]$
2.  **累加器 (Summator)** *(相当于连续系统中的积分器)*
    *   物理意义：将当前及过去所有的输入值加起来。
    *   冲激响应：$h[k] = u[k]$
3.  **差分器 (Diff / Differencer)** *(相当于连续系统中的微分器)*
    *   物理意义：输出当前输入与前一个输入之差。
    *   冲激响应：$h[k] = \delta[k] - \delta[k - 1]$
4.  **单位延迟器 (Unit delayer)**
    *   物理意义：把输入信号原封不动地向后延迟 1 个时刻。
    *   冲激响应：$h[k] = \delta[k - 1]$

---

## 三、 复合系统的冲激响应
与连续系统完全一致，离散系统的组合逻辑为：
1.  **级联系统 (Cascaded systems)**：串联在一起，冲激响应相**卷积**。
    $$h[k] = h_1[k] * h_2[k]$$
2.  **并联系统 (Parallel systems)**：并排在一起，冲激响应相**加**。
    $$h[k] = h_1[k] + h_2[k]$$

## 四、 📝【硬核例题推导：复合系统的冲激响应】
*(注意：PPT 第7页直接给出了最终答案，中间跳过了极为关键的卷积计算步骤，这里为你补全！)*

**【题目】** 求图中复合系统的单位冲激响应 $h[k]$。
已知各子系统：$h_1[k] = 2^k u[k]$，$h_2[k] = \delta[k-1]$，$h_3[k] = 3^k u[k]$，$h_4[k] = u[k]$。中间还有一条直通支路。

**【系统结构分析】**
1. 支路一（上方）：$h_1[k]$
2. 支路二（直通）：没有任何方框，等效于 $h_{直通}[k] = \delta[k]$
3. 支路三（下方）：$h_2[k]$ 与 $h_3[k]$ **级联**，即 $h_2[k] * h_3[k]$
前三条支路是**并联**关系，将它们相加后，再与最后的 $h_4[k]$ 发生**级联**（卷积）。

**【完整推导计算过程】**
**第一步：求出并联部分的总响应 $h_{inner}[k]$**
$$h_{inner}[k] = h_1[k] + \delta[k] + (h_2[k] * h_3[k])$$
其中，利用冲激函数的移位性质算级联：
$$h_2[k] * h_3[k] = \delta[k-1] * 3^k u[k] = 3^{k-1} u[k-1]$$
所以：$h_{inner}[k] = 2^k u[k] + \delta[k] + 3^{k-1} u[k-1]$

**第二步：整体与 $h_4[k]$ 进行卷积**
$$h[k] = h_{inner}[k] * h_4[k] = \Big( 2^k u[k] + \delta[k] + 3^{k-1} u[k-1] \Big) * u[k]$$
利用分配律，把这个大卷积拆成三项分别算：

*   **第 1 项**：$2^k u[k] * u[k]$
    *(利用上节课背诵的解析法公式：$\alpha^k u[k] * \beta^k u[k] = \frac{\beta^{k+1} - \alpha^{k+1}}{\beta - \alpha}u[k]$，这里 $\alpha=2, \beta=1$)*
    $= \frac{1^{k+1} - 2^{k+1}}{1 - 2} u[k] = (2^{k+1} - 1)u[k] = \mathbf{2 \cdot 2^k u[k] - u[k]}$
*   **第 2 项**：$\delta[k] * u[k]$
    *(任何信号与 $\delta[k]$ 卷积等于它本身)*
    $= \mathbf{u[k]}$
*   **第 3 项**：$3^{k-1} u[k-1] * u[k]$
    *(先算 $3^k u[k] * u[k]$，再利用移位特性)*
    $3^k u[k] * u[k] = \frac{1^{k+1} - 3^{k+1}}{1 - 3}u[k] = \frac{3^{k+1} - 1}{2}u[k]$
    将其整体右移 1 位（把 $k$ 换成 $k-1$）：
    $= \frac{3^{(k-1)+1} - 1}{2} u[k-1] = \frac{3^k - 1}{2} u[k-1] = \mathbf{[1.5(3)^{k-1} - 0.5] u[k-1]}$

**第三步：将三项结果相加**
$$h[k] = \big( 2 \cdot 2^k u[k] - u[k] \big) + \big( u[k] \big) + \big( [1.5(3)^{k-1} - 0.5] u[k-1] \big)$$
中间的 $-u[k]$ 和 $+u[k]$ 恰好抵消！
**最终结果**：
$$h[k] = 2(2)^k u[k] +[1.5(3)^{k-1} - 0.5]u[k-1]$$
*(完美对应了 PPT 第 7 页的最终公式！)*

## 五、 冲激响应与系统的因果性 (Causality)
*   **物理概念**：因果系统意味着“没有输入就没有输出”，系统的输出不会超前于输入出现（不能预知未来）。
*   **充要条件**：对于离散 LTI 系统，因果性的充分必要条件是冲激响应在 $k<0$ 时恒为 0：
    $$h[k] = 0, \quad k < 0$$

> **📝【判断例题】** 判断离散 LTI 系统 $h[k] = \cos[\frac{\pi}{2}k]u[k]$ 是否为因果系统。
> **解**：因为式子中乘了阶跃序列 $u[k]$，而当 $k<0$ 时 $u[k] = 0$。
> 因此必然满足当 $k<0$ 时 $h[k] = 0$。
> **结论**：这是一个因果系统。

---

## 六、 冲激响应与系统的稳定性 (Stability)
*   **物理概念**：如果系统对于任意的有界输入，都能产生有界的输出，则系统是稳定的 (BIBO 稳定)。
*   **充要条件**：离散 LTI 系统稳定的充分必要条件是，冲激响应绝对可和 (Absolutely summable)：
    $$\sum_{k=-\infty}^{\infty} |h[k]| = S < \infty$$

> **📝【判断例题】** 判断系统 $h[k] = 2^k \{ u[k] - u[k-3] \}$ 是否稳定。
> **解**：
> 1. 先搞清楚 $\{ u[k] - u[k-3] \}$ 是什么：它是一个矩形窗口，只在 $k=0, 1, 2$ 时值为 1，其余时刻都为 0。
> 2. 代入公式求绝对和：
>    $$\sum_{k=-\infty}^{\infty} |h[k]| = \sum_{k=0}^{2} |2^k| = 2^0 + 2^1 + 2^2 = 1 + 2 + 4 = 7$$
> 3. 因为 $7 < \infty$，满足绝对可和条件。
> **结论**：这是一个稳定系统。
