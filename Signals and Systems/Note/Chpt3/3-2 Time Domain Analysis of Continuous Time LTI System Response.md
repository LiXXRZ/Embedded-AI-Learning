# 连续时间 LTI 系统时域分析 学习笔记

## 一、 核心分析方法概览
连续LTI（线性时不变）系统的时域分析主要有三种途径：
1.  **经典法**：求解线性常系数微分方程。
2.  **分解法**：将响应分解为“零输入响应”与“零状态响应”。
3.  **卷积法**：利用脉冲响应 $h(t)$ 与输入 $x(t)$ 的卷积积分求解零状态响应。

---

## 二、 方法一：求解线性常系数微分方程 (经典法)

系统的输入输出关系描述为：
$$ y^{(n)}(t) + a_{n-1}y^{(n-1)}(t) + \dots + a_0 y(t) = b_m x^{(m)}(t) + \dots + b_0 x(t) $$

### 1. 全解的构成
$$ y(t) = y_h(t) + y_p(t) $$
*   **齐次解 (Homogeneous Solution) $y_h(t)$**:
    *   对应齐次方程（右边为0）。
    *   形式由**特征方程的根（特征根）**决定（只与系统本身特性有关，即模态）。
*   **特解 (Special/Particular Solution) $y_p(t)$**:
    *   形式由**激励信号 $x(t)$ 的形式**决定（强制分量）。

### 2. 典型例题解析 (Slides 4-6)
**题目**: 已知 $y''(t) + 6y'(t) + 8y(t) = x(t), t>0$。初始状态 $y(0)=1, y'(0)=2$，输入 $x(t)=e^{-t}u(t)$。

*   **步骤 1: 求齐次解 $y_h(t)$**
    *   特征方程: $s^2 + 6s + 8 = 0 \Rightarrow (s+2)(s+4)=0$
    *   特征根: $s_1 = -2, s_2 = -4$
    *   形式: $y_h(t) = A e^{-2t} + B e^{-4t}$

*   **步骤 2: 求特解 $y_p(t)$**
    *   根据输入 $x(t)=e^{-t}$，设特解为 $y_p(t) = C e^{-t}$。
    *   代入原方程: $C e^{-t} + 6(-C e^{-t}) + 8(C e^{-t}) = e^{-t}$
    *   解得 $3C = 1 \Rightarrow C = 1/3$。
    *   特解: $y_p(t) = \frac{1}{3} e^{-t}$

*   **步骤 3: 求全解并定系数**
    *   全解形式: $y(t) = A e^{-2t} + B e^{-4t} + \frac{1}{3} e^{-t}$
    *   代入初始条件 $y(0)=1, y'(0)=2$:
        *   $y(0) = A + B + 1/3 = 1$
        *   $y'(0) = -2A - 4B - 1/3 = 2$
    *   解得: $A = 5/2, B = -11/6$
    *   最终结果: $y(t) = \frac{5}{2}e^{-2t} - \frac{11}{6}e^{-4t} + \frac{1}{3}e^{-t}, \quad t>0$

### 3. 经典法的局限性 (Slide 9)
*   若输入信号改变，需重新求全解。
*   若初始条件改变，系数需重新计算。
*   **物理概念不清晰**: 无法区分哪部分响应是由初始储能产生的，哪部分是由外部输入产生的。

## 三、 方法二：响应的分解 (物理视角)

根据 LTI 系统的线性性质，系统全响应可以分解为：
$$ y(t) = y_{zi}(t) + y_{zs}(t) $$

### 1. 零输入响应 (Zero-Input Response, $y_{zi}$)
*   **定义**: 输入 $x(t)=0$ (没有外加激励)，仅由系统的**初始状态**产生的响应。
*   **物理模型**: 电容放电电路 (只有初始储能)。
*   **数学求解**:
    *   解齐次方程 $y^{(n)} + \dots + a_0 y = 0$。
    *   系数由初始条件完全确定。

### 2. 零状态响应 (Zero-State Response, $y_{zs}$)
*   **定义**: 初始状态为 0 (无初始储能)，仅由**外部输入 $x(t)$** 产生的响应。
*   **物理模型**: 电容充电电路 (初始电压为0)。
*   **数学求解**:
    *   利用卷积积分求解: $y_{zs}(t) = x(t) * h(t)$。

---

## 四、 零状态响应与卷积积分 (核心推导)

### 1. 推导思路 (Slides 14-15)
这是一个将信号分解再利用线性叠加的过程：

1.  **分解输入**: 将任意连续信号 $x(t)$ 看作无数个**窄脉冲**（矩形）的叠加。
    *   极限情况下，$\Delta \tau \to 0$，矩形变为冲激函数 $\delta(t)$。
    *   利用筛选性质: $x(t) = \int_{-\infty}^{+\infty} x(\tau)\delta(t-\tau) d\tau$。
2.  **利用 LTI 特性**:
    *   **冲激响应定义**: 输入 $\delta(t) \longrightarrow$ 输出 $h(t)$。
    *   **时不变性**: 输入 $\delta(t-\tau) \longrightarrow$ 输出 $h(t-\tau)$。
    *   **齐次性 (均匀性)**: 输入 $x(\tau)d\tau \cdot \delta(t-\tau) \longrightarrow$ 输出 $x(\tau)d\tau \cdot h(t-\tau)$。
    *   **叠加性**: 对所有时刻的响应积分。

### 2. 结论：卷积积分
$$ y_{zs}(t) = \int_{-\infty}^{+\infty} x(\tau)h(t-\tau) d\tau = x(t) * h(t) $$
*   物理意义：零状态响应 = 输入信号 与 冲激响应 的卷积。

---

## 五、 系统求解总结框图 (Slide 16)

求解连续 LTI 系统响应 $y(t)$ 的完整路径：

1.  **路径 A (求 $y_{zi}$)**:
    *   令输入 $x(t)=0$。
    *   根据初始状态求解齐次微分方程。

2.  **路径 B (求 $y_{zs}$)**:
    *   方法一：求解脉冲响应 $h(t)$，然后计算卷积 $x(t) * h(t)$。
    *   方法二：设初始状态为0，求解微分方程（通常用拉普拉斯变换辅助，本章侧重时域）。

3.  **合成**:
    $$ y(t) = y_{zi}(t) + y_{zs}(t) $$

## 复习重点提示
1.  **齐次解 vs 特解**: 掌握如何根据特征方程求齐次解，根据输入信号形式设特解。
2.  **概念区分**: 清楚 $y_h(t) + y_p(t)$ (数学解法) 与 $y_{zi}(t) + y_{zs}(t)$ (物理分解) 的区别。
    *   *注意：齐次解 $y_h$ 不完全等同于零输入响应 $y_{zi}$；全解中的自由分量包含 $y_{zi}$ 和部分 $y_{zs}$。*
3.  **卷积的来源**: 理解卷积积分是基于“信号分解为冲激脉冲”和“线性时不变特性”推导出来的。
