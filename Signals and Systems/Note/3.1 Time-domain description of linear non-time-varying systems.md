# LTI系统的时域描述 - 学习笔记

## 核心主题
本章节主要讨论如何使用**微分方程**（连续时间）和**差分方程**（离散时间）来描述线性时不变（LTI）系统的输入输出关系，并利用LTI系统的**微积分/差分求和特性**来求解响应。

---

## 一、 连续时间 LTI 系统的时域描述

### 1. 数学模型：线性常系数微分方程
连续LTI系统的物理模型（如电路系统）通常遵循物理定律（如KVL、KCL）。
*   **一阶系统**（如LR电路）：描述为一阶微分方程。
    *   $L\frac{di(t)}{dt} + Ri(t) = x(t)$
*   **二阶系统**（如LRC电路）：描述为二阶微分方程。
*   **一般形式**：
    输入 $x(t)$ 与输出 $y(t)$ 的关系由 **线性常系数微分方程 (Linear Constant Coefficient Differential Equation)** 描述：
    $$ \sum_{n} a_n \frac{d^n y(t)}{dt^n} = \sum_{m} b_m \frac{d^m x(t)}{dt^m} $$
    即：$y^{(n)}(t) + \dots + a_0 y(t) = b_m x^{(m)}(t) + \dots + b_0 x(t)$

### 2. 重要特性（解题技巧）
基于LTI系统的线性和时不变性，系统具有以下特性，可用于简化响应计算：

*   **微分特性 (Differential Property)**：
    若 $x(t) \longrightarrow y(t)$，则：
    $$ \frac{dx(t)}{dt} \longrightarrow \frac{dy(t)}{dt} $$
*   **积分特性 (Integral Property)**：
    若 $x(t) \longrightarrow y(t)$，则：
    $$ \int_{-\infty}^{t} x(\tau) d\tau \longrightarrow \int_{-\infty}^{t} y(\tau) d\tau $$

### 3. 典型例题分析
*   **例题1 (Slide 8)**：
    *   **已知**：输入 $x_1(t)$（阶跃信号）的响应为 $y_1(t)$。
    *   **求**：输入 $x_2(t)$（斜坡信号）的响应 $y_2(t)$。
    *   **思路**：观察发现 $x_2(t) = \int_{-\infty}^{t} x_1(\tau) d\tau$。
    *   **解法**：根据积分特性，$y_2(t) = \int_{-\infty}^{t} y_1(\tau) d\tau$。
*   **例题2 (Slide 9)**：
    *   **已知**：输入 $x_1(t)$（方波脉冲）的响应 $y_1(t) = \delta(t-1) - \delta(t-2)$。
    *   **求**：输入 $x_2(t)$（梯形波/三角波前段）的响应。
    *   **思路**：观察发现 $x_2(t)$ 是 $x_1(t)$ 的积分。
    *   **解法**：对 $y_1(t)$ 进行积分。$\int \delta(t-t_0) = u(t-t_0)$。
    *   **结果**：$y_2(t) = u(t-1) - u(t-2)$。

## 二、 离散时间 LTI 系统的时域描述

### 1. 数学模型：线性常系数差分方程
离散LTI系统的输入输出关系由**差分方程 (Difference Equation)** 描述。
*   **一般形式**：
    $$ \sum_{i=0}^{n} a_i y[k-i] = \sum_{j=0}^{m} b_j x[k-j] $$
    其中 $a_0=1$。
*   **分类**：包括前向差分（Forward）和后向差分（Backward）。对于因果系统，通常使用**后向差分方程**。

### 2. 典型应用：移动平均滤波器 (Moving Average Filter)
这是一个典型的离散LTI系统，常用于滤除信号噪声（平滑信号）。
*   **方程**（M点平均）：
    $$ y[k] = \frac{1}{M} \sum_{n=0}^{M-1} x[k-n] $$
*   **效果**：能够将混有高频噪声的信号变平滑（如Slide 16的MATLAB示例所示）。

### 3. 重要特性（解题技巧）
与连续系统类似，离散系统也具备相应的特性：

*   **差分特性 (Differential/Difference Property)**：
    若 $x[k] \longrightarrow y[k]$，则：
    $$ \nabla x[k] = x[k] - x[k-1] \longrightarrow \nabla y[k] = y[k] - y[k-1] $$
*   **求和特性 (Summation Property)**：
    若 $x[k] \longrightarrow y[k]$，则：
    $$ \sum_{n=-\infty}^{k} x[n] \longrightarrow \sum_{n=-\infty}^{k} y[n] $$

### 4. 典型例题分析 (Slide 18)
*   **已知**：输入 $x_1[k]$（阶跃序列）的响应为 $y_1[k]$。
*   **求**：输入 $x_2[k]$（脉冲序列 $\delta[k]$）的响应 $y_2[k]$。
*   **思路**：观察发现 $x_2[k] = \nabla x_1[k] = x_1[k] - x_1[k-1]$（脉冲是阶跃的差分）。
*   **解法**：利用差分特性，$y_2[k] = y_1[k] - y_1[k-1]$。
*   **计算细节**：对应点相减。
    *   $y_1 = \{1, 2, 3, 2, 1\}$
    *   $y_2[0] = 1-0=1$
    *   $y_2[1] = 2-1=1$
    *   $y_2[2] = 3-2=1$
    *   $y_2[3] = 2-3=-1$
    *   $y_2[4] = 1-2=-1$
    *   $y_2[5] = 0-1=-1$
    *   结果：$\{1, 1, 1, -1, -1, -1\}$

## 三、 总结与对比

| 特性 | 连续时间系统 (Continuous) | 离散时间系统 (Discrete) |
| :--- | :--- | :--- |
| **数学模型** | **微分方程** (Differential Eq) | **差分方程** (Difference Eq) |
| **基本运算** | 微分 $d/dt$，积分 $\int$ | 差分 $\nabla$，求和 $\sum$ |
| **解题策略** | 输入求导 $\to$ 输出求导<br>输入积分 $\to$ 输出积分 | 输入做差 $\to$ 输出做差<br>输入累加 $\to$ 输出累加 |
| **典型示例** | RL/LRC 电路 | 移动平均滤波器 (Moving Average) |

**复习建议**：
1.  熟记线性常系数微分/差分方程的通用形式。
2.  **重点掌握利用已知响应求未知响应的方法**：先判断输入信号之间的关系（是积分/求和关系，还是微分/差分关系），然后直接将此关系应用到输出信号上，避免重新解方程，能够极大提高解题效率。
