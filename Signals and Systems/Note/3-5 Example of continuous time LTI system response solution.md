# 学习笔记：连续LTI系统响应求解综合示例
**(An example of a continuous LTI system response solution)**

## 🌟 核心综合例题
**已知条件**：
描述某连续 LTI 系统的微分方程为：
$$y''(t) + 7y'(t) + 12y(t) = x(t), \quad t \ge 0$$
*   **激励信号 (Input)**：$x(t) = u(t)$
*   **初始状态 (Initial state)**：$y(0^-) = 1, \quad y'(0^-) = 2$

**求解目标**：
1. 零输入响应 $y_{zi}(t)$
2. 冲激响应 $h(t)$
3. 零状态响应 $y_{zs}(t)$
4. 完全响应 $y(t)$ 及其分量分解
5. 判断系统稳定性

---

### 第一步：求解零输入响应 $y_{zi}(t)$ (Zero-input response)
> **💡 概念**：零输入响应是指在没有外部激励（输入为0）的情况下，仅由系统的**初始状态**引起的响应。

1.  **列特征方程**：令微分方程左侧对应等于0。
    $$s^2 + 7s + 12 = 0$$
2.  **求特征根**：
    $$(s+3)(s+4) = 0 \implies s_1 = -3, \quad s_2 = -4$$
3.  **设齐次解的形式**：
    $$y_{zi}(t) = K_1 e^{-3t} + K_2 e^{-4t}, \quad t \ge 0^-$$
4.  **代入初始状态求系数**：
    将 $y(0^-)=1$ 和 $y'(0^-)=2$ 代入上式及其导数中：
    $$ \begin{cases} K_1 + K_2 = 1 \\ -3K_1 - 4K_2 = 2 \end{cases} \implies K_1 = 6, \quad K_2 = -5$$
5.  **得出结果**：
    $$y_{zi}(t) = 6e^{-3t} - 5e^{-4t}, \quad t \ge 0^-$$

>**细节一：零输入响应 $y_{zi}(t)$ 系数 $K_1, K_2$ 的求解过程**
>
>**已知：**
>齐次解的形式为：$y_{zi}(t) = K_1 e^{-3t} + K_2 e^{-4t}$
初始条件：$y_{zi}(0^-) = 1, \quad y'_{zi}(0^-) = 2$
>
>**详细计算步骤：**
>1. **对原式求导**：
   $$y'_{zi}(t) = -3K_1 e^{-3t} - 4K_2 e^{-4t}$$
>2. **将 $t=0$ 代入原式和导数式**（记住 $e^0 = 1$）：
   $$
   \begin{gathered}
   y_{zi}(0) = K_1 e^0 + K_2 e^0 \implies K_1 + K_2 = 1 \quad \text{—— (式1)} \\
   y'_{zi}(0) = -3K_1 e^0 - 4K_2 e^0 \implies -3K_1 - 4K_2 = 2 \quad \text{—— (式2)}
   \end{gathered}
   $$
>3. **解二元一次方程组**：
   由(式1)得到：$K_1 = 1 - K_2$
   代入(式2)：$-3(1 - K_2) - 4K_2 = 2$
   展开：$-3 + 3K_2 - 4K_2 = 2$
   合并：$-K_2 = 5 \implies \mathbf{K_2 = -5}$
   回代求 $K_1$：$K_1 = 1 - (-5) \implies \mathbf{K_1 = 6}$
>4. **最终结果**：
   $$y_{zi}(t) = 6e^{-3t} - 5e^{-4t}, \quad t \ge 0$$

---

### 第二步：求解冲激响应 $h(t)$ (Impulse response)
> **💡 概念**：系统在单位冲激信号 $\delta(t)$ 作用下，且初始状态为零时的响应。

1.  利用冲激平衡法（冲激函数匹配法），设 $h(t) = (A e^{-3t} + B e^{-4t})u(t)$。
2.  代入微分方程 $h''(t) + 7h'(t) + 12h(t) = \delta(t)$，利用 $t=0^+$ 时的跳变条件解出待定系数（过程略，由公式法可得 $A+B=0, -3A-4B=1$）。
3.  **得出结果**：
    $$h(t) = (e^{-3t} - e^{-4t})u(t)$$

>**细节二：利用“冲激平衡法”求冲激响应 $h(t)$ 的完整推导**
>
>**原方程：** $h''(t) + 7h'(t) + 12h(t) = \delta(t)$
由于右边是 $\delta(t)$，因此 $h(t)$ 必然在 $t=0$ 之后才有值，即带有 $u(t)$。
根据特征根 $s_1=-3, s_2=-4$，**设 $h(t) = (A e^{-3t} + B e^{-4t})u(t)$**。
>
>**详细求导与平衡步骤：**
>1. **求一阶导数 $h'(t)$**（利用乘积求导法则 $(uv)' = u'v + uv'$）：
   $$h'(t) = (-3A e^{-3t} - 4B e^{-4t})u(t) + (A e^{-3t} + B e^{-4t})u'(t)$$
   因为 $u'(t) = \delta(t)$，且根据冲激函数性质 $f(t)\delta(t) = f(0)\delta(t)$，代入 $t=0$：
   $$
   \begin{gathered}
   h'(t) = (-3A e^{-3t} - 4B e^{-4t})u(t) + (A e^0 + B e^0)\delta(t) \\
   h'(t) = (-3A e^{-3t} - 4B e^{-4t})u(t) + \mathbf{(A + B)\delta(t)}
   \end{gathered}
   $$
>
>2. **求二阶导数 $h''(t)$**：
   对上面的 $h'(t)$ 再次求导：
   $$h''(t) = (9A e^{-3t} + 16B e^{-4t})u(t) + (-3A e^{-3t} - 4B e^{-4t})\delta(t) + (A + B)\delta'(t)$$
   同样化简中间项：代入 $t=0$，得到 $(-3A - 4B)\delta(t)$。
   $$h''(t) = (9A e^{-3t} + 16B e^{-4t})u(t) + \mathbf{(-3A - 4B)\delta(t)} + \mathbf{(A + B)\delta'(t)}$$
>
>3. **代入原微分方程并“合并同类项”**：
   将求出的 $h''(t), h'(t), h(t)$ 统统代入 $h''(t) + 7h'(t) + 12h(t)$ 中。
>   * **$u(t)$ 前面的系数**（这部分必然等于0，因为 $A,B$ 是根据特征根设的）：
     $(9A - 21A + 12A)e^{-3t} + (16B - 28B + 12B)e^{-4t} = 0$ 
>   * **$\delta'(t)$ 前面的系数**：
     仅在 $h''(t)$ 中有，系数为 $\mathbf{(A + B)}$
>   * **$\delta(t)$ 前面的系数**：
     来自 $h''(t)$ 的 $(-3A - 4B)$ 和 $7 \times h'(t)$ 中的 $7(A + B)$。
     相加得：$-3A - 4B + 7A + 7B = \mathbf{4A + 3B}$
>
>4. **与等式右边匹配（这就是“平衡”）**：
   代入后的方程变为：$0 \cdot u(t) + (A+B)\delta'(t) + (4A+3B)\delta(t) = \mathbf{\delta(t)}$
   比较左右两边系数可得PPT上的神仙方程组：
>   * $\delta'(t)$ 的系数：**$A + B = 0$**
>   * $\delta(t)$ 的系数：**$4A + 3B = 1$**
>
>5. **解出 $A, B$**：
   $B = -A \implies 4A - 3A = 1 \implies \mathbf{A = 1}$
   从而 $\mathbf{B = -1}$。
   代回假设式，得到：$h(t) = (e^{-3t} - e^{-4t})u(t)$

### 第三步：求解零状态响应 $y_{zs}(t)$ (Zero-state response)
> **💡 概念**：系统在初始状态为零的情况下，仅由外部输入激励 $x(t)$ 引起的响应。它等于输入信号与冲激响应的**卷积**。

1.  **利用卷积积分计算**：
    $$y_{zs}(t) = x(t) * h(t) = u(t) * [(e^{-3t} - e^{-4t})u(t)]$$
2.  **得出结果**（直接积分推导得）：
    $$y_{zs}(t) = \left(\frac{1}{12} - \frac{1}{3}e^{-3t} + \frac{1}{4}e^{-4t}\right)u(t)$$

>**细节三：零状态响应 $y_{zs}(t)$ 的卷积积分求解**
>
>公式为：$y_{zs}(t) = x(t) * h(t) = \int_{-\infty}^{\infty} x(t-\tau)h(\tau) d\tau$
已知输入 $x(t) = u(t)$，冲激响应 $h(t) = (e^{-3t} - e^{-4t})u(t)$
>
>**详细积分步骤：**
>1. **写出积分表达式**：
   $$y_{zs}(t) = \int_{-\infty}^{\infty} u(t-\tau) \cdot \left[ (e^{-3\tau} - e^{-4\tau}) u(\tau) \right] d\tau$$
>
>2. **确定积分上下限（破除 $u(t)$）**：
>   * 因为有 $u(\tau)$，所以 $\tau > 0$（确定了**积分下限为 0**）。
>   * 因为有 $u(t-\tau)$，所以 $t - \tau > 0 \implies \tau < t$（确定了**积分上限为 $t$**）。
>   * 只有当 $t > 0$ 时，上限才大于下限，积分才有意义。所以积分结果最后要乘上一个 $u(t)$。
>
>3. **执行定积分计算**：
   $$y_{zs}(t) = \left( \int_{0}^{t} (e^{-3\tau} - e^{-4\tau}) d\tau \right) \cdot u(t)$$
   分别对两项进行积分：
   $$= \left[ \frac{1}{-3}e^{-3\tau} - \frac{1}{-4}e^{-4\tau} \right]_0^t \cdot u(t)$$
   $$= \left[ \left(-\frac{1}{3}e^{-3t} + \frac{1}{4}e^{-4t}\right) - \left(-\frac{1}{3}e^0 + \frac{1}{4}e^0\right) \right] \cdot u(t)$$
   计算常数项：$-\frac{1}{3}e^0 + \frac{1}{4}e^0 = -\frac{1}{3} + \frac{1}{4} = -\frac{4}{12} + \frac{3}{12} = -\frac{1}{12}$
>   
>4. **得出最终结果**：
   $$= \left( -\frac{1}{3}e^{-3t} + \frac{1}{4}e^{-4t} - \left( -\frac{1}{12} \right) \right) \cdot u(t)$$
   将常数项移到最前面，完全对应PPT结果：
   $$y_{zs}(t) = \left( \frac{1}{12} - \frac{1}{3}e^{-3t} + \frac{1}{4}e^{-4t} \right) u(t)$$

---

### 第四步：求解完全响应 $y(t)$ 及其分解 (Complete response)
**完全响应 = 零输入响应 + 零状态响应**
$$y(t) = y_{zi}(t) + y_{zs}(t) \\
= (6e^{-3t} - 5e^{-4t}) + \left(\frac{1}{12} - \frac{1}{3}e^{-3t} + \frac{1}{4}e^{-4t}\right)$$

合并同类项得到完全响应：
$$y(t) = \frac{1}{12} + \frac{17}{3}e^{-3t} - \frac{19}{4}e^{-4t}, \quad t > 0$$

#### 🔍 响应分量的两种物理分解方式（必考考点）
课件中展示了对完全响应 $y(t)$ 的两种拆解视角：

1. **按根的来源分解：固有响应 (Intrinsic) vs 强迫响应 (Forced)**
   *   **固有响应 $y_h(t)$**：由系统本身的特性（特征根）决定的部分，即含有 $e^{-3t}$ 和 $e^{-4t}$ 的项。
       $$y_h(t) = \frac{17}{3}e^{-3t} - \frac{19}{4}e^{-4t}$$
   *   **强迫响应 $y_p(t)$**：形式由外部输入信号决定的部分（此处输入为阶跃，对应常数项）。
       $$y_p(t) = \frac{1}{12}$$

2. **按随时间演变的趋势分解：暂态响应 (Transient) vs 稳态响应 (Steady-state)**
   *   **暂态响应 $y_t(t)$**：随着时间 $t \to \infty$ 会衰减到 0 的部分。
       $$y_t(t) = \frac{17}{3}e^{-3t} - \frac{19}{4}e^{-4t}$$
   *   **稳态响应 $y_s(t)$**：随着时间 $t \to \infty$ 不趋于 0，稳定保留下来的部分。
       $$y_s(t) = \frac{1}{12}$$

---

### 第五步：判断系统稳定性 (Determine system stability)
利用第一份笔记中提到的**绝对可积**条件：
$$\int_{-\infty}^{\infty} |h(\tau)| d\tau = \int_{0}^{\infty} (e^{-3\tau} - e^{-4\tau}) d\tau = \left( \frac{1}{3} - \frac{1}{4} \right) = \frac{1}{12} < \infty$$
因为冲激响应绝对可积（结果为有限值），因此该连续 LTI 系统是**稳定系统 (Stable System)**。

---

### 第六步：拓展应用——LTI系统特性的应用
**新问题**：如果初始状态不变，但输入信号变成了 $x_1(t) = 0.5u(t-1)$，求系统的响应。

> **📝 易错提醒与解题思路**：
> 1.  **零输入响应 $y_{zi}(t)$ 不变**：因为它只与初始状态有关，与外部输入无关！
> 2.  **零状态响应利用LTI特性**：新输入 $x_1(t) = 0.5 x(t-1)$ 是原输入的“缩放+延时”。根据系统的**线性 (Linear)** 和 **时不变性 (Time-invariant)**，新的零状态响应只需对原来的零状态响应进行同样的“缩放+延时”即可。

**计算过程**：
1.  **新零输入响应**：保持不变。
    $$y_{zi}(t) = 6e^{-3t} - 5e^{-4t}, \quad t \ge 0^-$$
2.  **新零状态响应**：将原 $y_{zs}(t)$ 乘以0.5，并将所有 $t$ 替换为 $(t-1)$。
    $$y_{1zs}(t) = 0.5 y_{zs}(t-1) = 0.5 \left( \frac{1}{12} - \frac{1}{3}e^{-3(t-1)} + \frac{1}{4}e^{-4(t-1)} \right) u(t-1)$$
    展开得：
    $$y_{1zs}(t) = \left( \frac{1}{24} - \frac{1}{6}e^{-3(t-1)} + \frac{1}{8}e^{-4(t-1)} \right) u(t-1)$$
3.  **新完全响应**：
    $$y(t) = y_{zi}(t) + y_{1zs}(t)$$
